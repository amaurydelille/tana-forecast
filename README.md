# Tana Forecast
A decoded only time-series foundation model trained to outperformed current other time-series SOTA models.

## Current Status

![Tana Forecast Architecture](assets/tana-forecast.drawio.png)

The model currently has 15M parameters and is being trained of various stocks, logistic and natural events datasets.
We chose a default 4096 data points context window and 256 data points prediction length.
